# AI自动配图功能使用说明

## 功能概述

系统已集成**通义千问文生图API**，可以根据文章内容自动生成配图，提供AI自动配图功能。

## 功能特点

- ✅ **智能提示词生成**：自动根据文章内容提取主题，生成适合的绘画提示词
- ✅ **高质量图片**：使用 qwen-image-max 模型，生成高质量配图
- ✅ **多种尺寸**：支持 16:9、4:3、1:1 三种图片比例
- ✅ **批量生成**：一次可生成多张配图，默认3张
- ✅ **自动优化**：启用 prompt_extend，自动优化提示词质量
- ✅ **无水印**：生成的图片不添加水印，可直接使用

## API配置

### 确保API Key已配置

打开 `backend/.env` 文件，确保已配置通义千问API Key：

```bash
DASHSCOPE_API_KEY=sk-你的API密钥
DASHSCOPE_MODEL=qwen-plus
```

如果还没有API Key，请访问 [阿里云百炼平台](https://bailian.console.aliyun.com/) 获取。

## 使用方法

### 方式一：通过API调用

在请求的 `options` 中添加以下参数：

```json
{
  "contentUrl": "https://mp.weixin.qq.com/s/...",
  "styleUrl": "https://mp.weixin.qq.com/s/...",
  "options": {
    "type": "expand",
    "style": "casual",
    "useAIImages": true,
    "aiImageCount": 3
  }
}
```

**参数说明：**

- `useAIImages`: 是否使用AI自动配图（布尔值，true/false）
- `aiImageCount`: AI配图数量（数字，默认3张，建议1-5张）

### 方式二：通过前端界面（如果前端已集成）

1. 打开公众号复刻页面
2. 输入内容链接和样式参考链接
3. 勾选"使用AI自动配图"选项
4. 设置配图数量（默认3张）
5. 点击"生成"

系统会自动：
- ✅ 提取文章主题
- ✅ 生成绘画提示词
- ✅ 调用通义千问文生图API
- ✅ 返回生成的图片URL

## API限制

- **图片尺寸**：
  - 1664*928 (16:9，横图，默认)
  - 1472*1104 (4:3，标准)
  - 1328*1328 (1:1，方形)
- **提示词长度**：≤800字符
- **负面提示词**：≤500字符
- **图片有效期**：24小时（阿里云OSS临时URL）
- **生成时间**：约5-15秒/张
- **并发限制**：建议间隔1秒以上

## 注意事项

1. **图片有效期**：
   - 生成的图片URL有效期仅为24小时
   - 建议在使用时立即下载到本地或上传到永久图床

2. **网络要求**：
   - 需要能访问阿里云API（dashscope.aliyuncs.com）
   - 图片托管在阿里云OSS，需要网络连接

3. **API费用**：
   - 文生图API调用会产生费用
   - 建议控制生成数量，避免不必要的费用

4. **图片质量**：
   - qwen-image-max 模型质量最高
   - 可以通过调整提示词来控制图片风格

5. **内容安全**：
   - 自动过滤不合规内容
   - 避免生成涉及敏感话题的图片

## 常见问题

### Q: AI配图和原文配图如何选择？

A:
- **使用原文配图**：如果原文图片质量好、数量合适，且不需要担心防盗链问题
- **使用AI配图**：如果需要原创配图、想要特定风格，或者原文图片不合适

### Q: 图片URL过期了怎么办？

A: 需要在24小时内下载图片并上传到永久图床。系统未来可能集成自动下载功能。

### Q: 如何提高AI配图质量？

A:
1. 优化文章内容，主题更明确
2. 调整 `aiImageCount`，生成更多选择
3. 手动指定更精确的提示词（需要修改代码）

### Q: 生成图片失败怎么办？

A: 检查：
1. API Key是否正确配置
2. 网络是否能访问阿里云API
3. API余额是否充足
4. 提示词是否符合规范（≤800字符）

## 代码示例

### 前端调用示例

```typescript
const response = await fetch('http://localhost:3001/api/recreate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    contentUrl: 'https://mp.weixin.qq.com/s/...',
    styleUrl: 'https://mp.weixin.qq.com/s/...',
    options: {
      type: 'expand',
      style: 'casual',
      useAIImages: true,  // 启用AI配图
      aiImageCount: 3     // 生成3张图片
    }
  })
});

const result = await response.json();
console.log('AI生成的图片:', result.data.images);
```
