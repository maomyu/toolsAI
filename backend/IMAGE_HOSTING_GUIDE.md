# 图床上传功能使用说明

## 功能概述

系统已集成**SM.MS免费图床**，可以自动将微信公众号的图片上传到图床，解决防盗链问题。

## 使用步骤

### 1. 注册SM.MS账号

1. 访问 [https://sm.ms/](https://sm.ms/)
2. 注册账号（免费）
3. 登录后进入 Dashboard
4. 生成并复制 **Secret Token**

### 2. 配置API Token

打开 `backend/.env` 文件，添加你的Token：

```bash
SMMS_API_TOKEN=你的Token
```

例如：
```bash
SMMS_API_TOKEN=abcd1234efgh5678ijkl9012mnop3456
```

### 3. 重启后端服务器

```bash
cd backend
npm run dev
```

### 4. 使用功能

1. 打开公众号复刻页面
2. 输入内容链接和样式参考链接
3. 确保选择了"使用原文配图"
4. 点击"生成"

系统会自动：
- ✅ 从原文提取图片
- ✅ 上传到SM.MS图床
- ✅ 替换为新的公开访问URL
- ✅ 插入到二创文章中

## API限制

- **单文件大小**：最大 5MB
- **免费用户**：
  - 每小时上传限制：根据SM.MS政策
  - 存储时间：永久
- **响应时间**：通常 1-3秒/张

## 注意事项

1. **网络要求**：需要能访问 SM.MS 服务器
2. **API Token**：请妥善保管，不要泄露
3. **未配置Token**：如果未配置，系统会跳过上传，直接使用原图URL
4. **上传失败**：如果某张图片上传失败，会使用原URL，不影响其他图片

## 优势

- ✅ **解决防盗链**：图片可以在任何地方正常显示
- ✅ **稳定可靠**：SM.MS是知名免费图床，访问稳定
- ✅ **自动化**：无需手动上传，系统自动处理
- ✅ **永久存储**：上传的图片永久保存在图床
- ✅ **CDN加速**：SM.MS提供CDN加速，访问速度快

## 常见问题

### Q: 上传失败怎么办？
A: 检查：
1. API Token是否正确配置
2. 网络是否能访问 sm.ms
3. 图片大小是否超过5MB

### Q: 图片显示慢？
A: SM.MS提供全球CDN，第一次加载可能稍慢，之后会缓存

### Q: 不想使用图床怎么办？
A: 清空 `.env` 文件中的 `SMMS_API_TOKEN` 配置即可
